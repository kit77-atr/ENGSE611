📊 Performance Monitoring & DocumentFragment Demo

ตัวอย่างการวัดประสิทธิภาพด้วย console.time() และการใช้ DocumentFragment เพื่อเพิ่ม DOM จำนวนมากโดยไม่กระตุก

1) การวัดเวลา (Benchmarking) ด้วย console.time()

กดปุ่มเพื่อทดสอบความเร็วของ Loop 1 ล้านครั้ง

// JavaScript ที่ใช้ทดสอบ console.time()

console.time('Loop 1M');
let sum = 0;
for (let i = 0; i < 1000000; i++) {
  sum += i;
}
console.timeEnd('Loop 1M');

2) DocumentFragment (ช้า vs เร็ว)

ตัวอย่างการเพิ่ม li จำนวนมาก เพื่อเปรียบเทียบ:

  • Normal DOM: เพิ่มทีละชิ้น → ช้า
  • Fragment DOM: รวมใน fragment ก่อน → เร็วมาก

      // โค้ดทดสอบ Normal DOM vs Fragment DOM

// Normal DOM
console.time('Normal DOM');
for (let i = 0; i < 1000; i++) {
  const li = document.createElement('li');
  li.textContent = `Item ${i}`;
  list1.appendChild(li);
}
console.timeEnd('Normal DOM');

// Fragment DOM
console.time('Fragment DOM');
for (let i = 0; i < 1000; i++) {
  const li = document.createElement('li');
  li.textContent = `Item ${i}`;
  fragment.appendChild(li);
}
list2.appendChild(fragment);
console.timeEnd('Fragment DOM');
